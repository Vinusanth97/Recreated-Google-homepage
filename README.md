# Google Homepage Clone (Tailwind CSS)

This is a simple Google homepage clone built using HTML and Tailwind CSS.

## 🚀 How to Run

### 1. Clone the project
git clone https://github.com/Vinusanth97/Recreated-Google-homepage.git

### 2. Go to project folder
cd Recreated-Google-homepage

### 3. Install dependencies
npm install

### 4. Run Tailwind
npx @tailwindcss/cli -i ./src/input.css -o ./src/output.css --watch

### 5. Open the project
Open this file in your browser:
src/index.html


## 🛠️ Tech Used

* HTML
* Tailwind CSS
